Data Format (Response):

[

        "name": "John Doe",

        "image": "path/to/image.jpg"

GET /api/Drivers/{id}

Hent detaljer om en spesifikk sjåfør ved ID.

[HttpGet("{id}")]
public async Task<ActionResult<Driver>> Get(int id)

GET /api/Drivers/byName/{name}

Hent detaljer om en spesifikk sjåfør ved navn.

[HttpGet("byName/{name}")]
public async Task<ActionResult<Driver>> GetByName(string name)

POST /api/Drivers

Opprett en ny sjåfør.

[HttpPost]
public async Task<ActionResult<Driver>> Post(Driver driver)

PUT /api/Drivers/{id}

Oppdater detaljer om en spesifikk sjåfør ved ID.

[HttpPut("{id}")]
public async Task<ActionResult<Driver>> Put(int id, Driver driver)

DELETE /api/Drivers/{id}

Slett en spesifikk sjåfør ved ID.

[HttpDelete("{id}")]
public async Task<ActionResult<Driver>> Delete(int id)

[HttpPost]
public IActionResult PostImage(IFormFile formFile)

Data Format (Response):

Upon successful image upload, returns HTTP 200 OK.
